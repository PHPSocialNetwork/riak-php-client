Class QueryIndex

Used to query a secondary index in Riak.

$command = (new Command\Builder\QueryIndex($riak))
  ->buildBucket('users')
  ->withIndex('users_name', 'Knuth')
  ->build();

$response = $command->execute($command);

$index_results = $response->getIndexResults();
